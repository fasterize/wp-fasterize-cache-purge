== Description ==
Plugin for refresh Wordpress pages on the Fasterize cache.

Fasterize Cache Purge sends a refresh order to the URL of a page or post every time it is modified. This occurs when editing, publishing, commenting or deleting an item.

== Requirements ==
This plugin is useful only if Fasterize is configured to force the caching of the optimized pages of the website.
By default, Fasterize will follow the cache headers sent by your server.

Contact support@fasterize.com for more information.

== Installation ==
Download to your plugin directory. Or simply install via Wordpress admin interface.
Activate.
